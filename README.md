# Interactive AI Engineer & Developer Portfolio

A professional, modern, and highly interactive portfolio website built for **Nikhil Chowdary**, an aspiring AI Engineer, Python Developer, and Full-Stack Learner. 

🚀 **Live Demo:** [https://nikhilchowdary-45.github.io/portfolio/](https://nikhilchowdary-45.github.io/portfolio/)

---

## ✨ Features

* **🖥️ Deep Space Gothic Theme**: Built using a modern dark aesthetic with neon yellow highlights, glassmorphic floating elements, and premium typography.
* **🌓 Theme Toggle (Dark & Light Mode)**: Fully integrated theme switcher button in the header that remembers preferences locally (`localStorage`).
* **🌌 Interactive Particle Canvas Network**: A physics-based canvas background that spawns floating nodes connecting to the user's cursor.
* **✨ Inertial Trail Cursor**: A custom circular cursor trail that smoothly follows the mouse pointer and expands over interactive elements.
* **🎨 3D Card Hover Tilt Effects**: High-performance mathematical tilt transforms that dynamically rotate project, education, and credential cards in 3D space based on mouse position.
* **🎓 Education History & Milestones**: Horizontal grid layouts highlighting expected B.Tech, Intermediate MPC, and SSC graduation parameters.
* **📜 Licenses & Certifications Column**: A three-column grid showcasing vector previews of certifications (AI Engineering, Python Programming, and Full-Stack Web Development) with one-click view access.
* **✉️ Simulated Packet Form Submissions**: Contact page with animated inputs and a stateful dispatcher simulation indicating loading, success, and reset states.

---

## 🛠️ Tech Stack

* **Structure & Markup**: HTML5
* **Design & Styling**: Custom CSS3 (Vanilla variable parameters, Glassmorphism, animations, responsive grid flex layouts)
* **Logic & Interactions**: Modern JavaScript (ES6+, Canvas Physics API, 3D inertia coordinate tracker, scroll spyers, local state managers)
* **Fonts & Icons**: Outfit & Space Grotesk (Google Fonts), FontAwesome Icons v6

---

## 📂 File Architecture

* `index.html` - Semantic markup structure and core section layouts
* `index.css` - Custom variable stylesheets, light/dark themes, card tilt matrices, and media query breakpoints
* `index.js` - Canvas node drawing engine, inertia cursor trails, category filter switches, and theme switch listeners
* `profile.jpg` - Inline profile headshot
* `resume.jpg` - Downloadable curriculum vitae page
* `cert_ai.svg`, `cert_python.svg`, `cert_fullstack.svg` - Vector certificate graphics

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nikhilchowdary-45/portfolio.git
   cd portfolio
   ```
2. **Launch a local server:**
   You can run a local server using Python, Node, or VS Code's Live Server:
   ```bash
   # Using Python
   python -m http.server 8080
   ```
3. **Open in browser:**
   Go to `http://localhost:8080/index.html` to preview the site.
